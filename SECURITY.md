# Politique de sécurité

## Signaler une vulnérabilité

Merci de **ne pas ouvrir d'issue publique** pour une faille de sécurité.

Le signalement privé est activé sur ce dépôt. Rends-toi dans l'onglet
**Security → Report a vulnerability**, ou directement :

<https://github.com/double-v/libre/security/advisories/new>

Le rapport reste confidentiel entre toi et le mainteneur jusqu'à la
publication d'un correctif.

Merci d'inclure, si possible :

- une description de la faille et de son impact ;
- les étapes pour la reproduire ;
- la version ou le commit concerné.

Nous faisons de notre mieux pour accuser réception sous 72 h.

## Périmètre

Ce dépôt héberge « Libre », une application de rencontre. Les rapports
concernant l'exposition de données personnelles, l'authentification,
l'autorisation, les injections ou le contournement de la modération sont
particulièrement bienvenus.
