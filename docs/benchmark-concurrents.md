## Détail par application

### Tinder
- **Plus** : ~15 €/mois | **Gold** : ~20 €/mois | **Platinum** : ~40 €/mois
- Super Like : ~5 €/unité | Boost : micro-paiement
- Tarification dynamique (selon âge, localisation, historique)
- Sans abonnement : likes limités, impossible de voir qui vous a liké
- Coût annuel estimé utilisateur actif : **300-480 €**

### Bumble
- **Boost** : ~13-19 €/mois | **Premium** : ~33 €/mois
- Offre à vie : 190 € (ponctuel)
- Compliments : ~2 €/unité
- Sans abonnement : swipes limités, les femmes ont 24h pour initier
- Coût annuel estimé utilisateur actif : **230-400 €**

### Happn
- **Premium** : ~11-25 €/mois (selon engagement)
- CrushTime : ~5 €/unité
- Sans abonnement : FlashNotes limitées, pas de "qui vous a liké"
- Coût annuel estimé utilisateur actif : **120-300 €**

### Pure
- Abonnement obligatoire : ~18 €/mois (1 semaine à 10 €)
- Compliment : ~3 €/unité | King of the Hill (boost) : ~3,50 €/unité | Incognito : ~3 €
- Version gratuite : **essai 3 jours**, puis payant obligatoire
- Coût annuel estimé utilisateur actif : **216-500 €** (abonnement + microtransactions)

### Fruitz
- **Premium** : ~5-12 €/mois | **Golden** : ~11-30 €/mois
- Golden à vie : 210 €
- Version gratuite : 30 likes / 12h
- Coût annuel estimé utilisateur actif : **65-360 €**

## Faits marquants pour la communication

1. **47 % des abonnés regrettent** leur achat dans le mois (UFC-Que Choisir 2025)
2. Un utilisateur actif sur une app premium dépense en moyenne **200-400 €/an**
3. La version gratuite de la plupart des apps est bridée (likes limités, fonctionnalités verrouillées) — conçue pour pousser à l'abonnement
4. Les microtransactions (boosts, super likes, compliments) s'additionnent facilement : +50-150 €/an
5. La tarification est dynamique : deux personnes voient des prix différents pour le même abonnement
6. Le renouvellement automatique est activé par défaut

## Argumentaire Libre

- **Gratuit = plus de célibataires présents** — Pas de barrière financière = plus large ratisseage = plus de chances de connexion
- **Pas de fonctionnalités verrouillées** — Tout est accessible à tout le monde
- **Pas de microtransactions** — Pas de boost à acheter, pas de super like en option
- **Le chat est le pont, pas le produit** — On ne cherche pas à vous garder en ligne
- **Chiffrement E2E** — Vos messages, le serveur ne les lit pas
